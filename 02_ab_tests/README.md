# A/B тесты (2 мини-проекта)

В этой папке лежат два A/B теста, оформленные как отдельные проекты

## 1) A/B по непрерывной метрике (выручка)
Папка: **ab_continuous_revenue**

Что сделано:
- чтение данных, базовые проверки качества
- сравнение средних (эффект = средняя в test − средняя в control)
- t-тест и p-value
- 95% доверительный интервал для эффекта
- проверка влияния выбросов (сравнение с p99 / “основной массой”)

Результат и выводы — внутри README и ноутбука проекта

## 2) A/B по долям (конверсия)
Папка: **ab_proportions_conversion**

Что сделано:
- проверка корректности эксперимента (группа ↔ страница), чистка данных
- удаление дублей пользователей (1 пользователь = 1 наблюдение)
- расчёт конверсии по группам и эффекта (delta)
- z-тест для долей и p-value
- 95% доверительный интервал для delta
- проверки устойчивости по дням (daily conversion / daily delta / daily n)

Результат и выводы — внутри README и ноутбука проекта

## Как открыть
Внутри каждого проекта есть:
- **notebooks** — ноутбук с расчётами
- **data** — исходный датасет
- **figures** — графики

Открыть ноутбук из **notebooks** и запустить по порядку
